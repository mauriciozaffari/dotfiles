# Domain & Trademark Research Guide

This file provides practical guidance on checking domain availability and trademark risk for name candidates.

---

## Domain Strategy

### Priority Order for Domains

1. **`[name].com`** — Always the gold standard. Worth acquiring if under $5K for most startups.
2. **`get[name].com`** — Common and accepted (e.g., getdropbox.com was Dropbox's early domain)
3. **`[name].io`** — Widely used and respected in tech
4. **`[name].co`** — Acceptable, especially for startups
5. **`[name]hq.com`** or **`[name]app.com`** — Functional fallbacks
6. **Country TLDs** (`.ai`, `.vc`, `.so`) — Only if it adds brand meaning (e.g., `.ai` for AI company)

### Domain Availability Rules of Thumb

- Single common English words: almost certainly taken on `.com`
- Two-word combinations: 50/50 on `.com` depending on specificity
- Invented/coined words: much higher availability
- If `.com` is taken, check who owns it — many are parked domains that can be acquired

### Domain Red Flags

- Domain is actively used by a competitor or similar business — avoid the name
- Domain is held by a domain squatter — can often be acquired for $500–$5K
- Domain redirects to a shady site — requires more due diligence

## Trademark Research

### Why It Matters

A name can be available as a domain but still be trademarked — and building a brand on a trademarked name is a serious legal and financial risk.

### Free First-Pass Checks

- **USPTO TESS** (US): https://www.uspto.gov/trademarks/search
- **EUIPO** (Europe): https://euipo.europa.eu/eSearch/
- **WIPO Global Brand Database** (International): https://branddb.wipo.int/

### What to Search

1. Search the exact name
2. Search phonetic variants (e.g., "Fiverr" → also search "Fiver", "Five-r")
3. Search in your specific product/service category (trademark classes matter — "Apple" can be a computer brand AND a record label)

### Risk Levels

| Situation | Risk Level | Recommendation |
|---|---|---|
| Name not found in any search | Low | Proceed, but get legal review before launch |
| Name found in a different industry | Medium | Consult a trademark attorney |
| Name found in your industry | High | Do not use — find an alternative |
| Name is descriptive (generic) | Medium | Generic terms are hard to trademark, you AND competitors can't own it |

### Trademark Classes Most Relevant for Startups

- **Class 35**: Business services, SaaS, marketing
- **Class 36**: Financial services, fintech
- **Class 38**: Telecommunications, messaging, social platforms
- **Class 42**: Software as a service, tech development

### Key Principle

**A domain check is not a trademark check.** Always do both. Getting a trademark attorney to do a clearance search ($300–$800) before committing to a name is almost always worth it.

## Social Handle Availability

Check availability on:
- Instagram, TikTok, LinkedIn, YouTube, X (Twitter)

Tools that help:
- **Namecheckr.com** — checks availability across 30+ platforms simultaneously
- **Knowem.com** — broader platform coverage

### Handle Strategy When Exact Match Is Taken
- `[name]app`, `[name]hq`, `get[name]`, `[name]official`
- Consider whether the brand can live with an imperfect handle — sometimes yes

## International Checks

For brands targeting global markets, run the name through:

1. **Google Translate** for Spanish, French, Mandarin, Arabic, Portuguese, German
2. Ask a native speaker in target markets if the name has any slang or negative connotation
3. Check pronunciation difficulty — names with hard consonant clusters don't travel well

### Famous International Naming Failures (Learn From These)
- Chevrolet **Nova** flopped in Spanish-speaking markets ("no va" = "doesn't go")
- Mitsubishi **Pajero** means something obscene in Spanish slang
- Clairol **Mist Stick** — "mist" is slang for manure in German

## The Bottom Line Checklist Before Committing to a Name

- [ ] Domain secured or path to secure it identified
- [ ] Trademark search done on USPTO (and WIPO if global)
- [ ] Social handles checked
- [ ] Name tested with native speakers in primary target markets
- [ ] Legal clearance search done (or scheduled)
- [ ] Team alignment — key stakeholders aligned before announcing
